# Plugin Icons

This directory contains SVG icons for the Contexture Music Player plugin.

## Required Icons

- `music_player.svg` - Main plugin icon (13x13 for tool window)
- `play_pause.svg` - Play/pause button icon
- `stop.svg` - Stop button icon
- `next.svg` - Next track button icon
- `previous.svg` - Previous track button icon

## Icon Guidelines

- Use SVG format for scalability
- Follow IntelliJ Platform icon guidelines
- Support both light and dark themes
- Size: 13x13 pixels for tool window icons, 16x16 for action icons

## Temporary Placeholders

For now, the plugin will use default IntelliJ icons. Custom icons will be added later.
